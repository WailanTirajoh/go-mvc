# GO Simple clean architecture

Opensource Package:
- GORM
- VIPER
- GORILLA MUX
- MYSQL

## Instalation Step
```
#clone this repo
git@github.com:WailanTirajoh/go-simple-clean-architecture.git

#create .env from .example
cp .env.example .env

#create database
mysql -u root -p

create database 2022_godb;
exit;

go get .
go run .
```
